Explanation for csv-data files: 
pulse_amplitude_[name]_arr: 1d array, pulse amplitude for the in-phase component of a R_X(pi/2)-gate as a function of gate duration. (units: V)
duration_[name]_arr: 1d array, gate durations used in experiments (units: s)

Here, the name is the pulse name from the set {FAST_DRAG-L, HD_DRAG-L, Cosine_DRAG-L}.
