Explanation for csv-data files: 
e_state_probability_vs_Nr_and_td_mat: 2d array, excited state probability from a cross-quadrature distortion characterization experiment as a function of the number of repeated (R_X(pi/2), R_X(pi/2), R_X(-pi/2), R_X(-pi/2))-sequences (rows) and the delay between pulses (columns). 
number_of_repeats_arr: 1d array, number of repeated (R_X(pi/2), R_X(pi/2), R_X(-pi/2), R_X(-pi/2))-sequences used in the experiment.
delay_arr: 1d array, delay between pulses used in the experiment.  
pulse_duration: 1-element array containing the used pulse duration 

Here, we used a FAST DRAG-P pulse with envelope predistortion.
